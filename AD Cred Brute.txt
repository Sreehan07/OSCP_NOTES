#### custom list of usernames
Ruby-based tool [Username Anarchy](https://github.com/urbanadventurer/username-anarchy) to convert a list of real names into common username formats.
```shell-session
./username-anarchy -i /home/ltnbob/names.txt 
```

#### Enumerating valid usernames with Kerbrute
do validate which username present form the custom list generated by anarchy
```shell-session
./kerbrute_linux_amd64 userenum --dc 10.129.201.57 --domain inlanefreight.local names.txt
```
